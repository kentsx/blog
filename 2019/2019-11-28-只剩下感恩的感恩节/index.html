只剩下感恩的感恩节

2019-11-28
K与J的爱情

来源：https://www.douban.com/note/743648246/

自从有了自己的站点以后，也不太想到豆瓣上来写写画画了。但毕竟自己的站点确实太粗糙了，所以有时候使用起来真的还不如豆瓣来的痛快。只不过，自己的站，我可以信马由缰，豆瓣，发发“骚客”的骚气就足够了。

今天是中国时间的感恩节。本来我是不知道的，早上在公交车上，手机推送进来，匆忙间扫视到了感恩节几个字眼，平时都不怎么敏锐的眼睛，今天像是有预兆一样，一定要让我清晰明白地看到那几个字。很刺眼，很刺心。

原因很简单，去年这个时候，我正在美国，吃着朋友的感恩节大餐，准备着一年一度的黑五狂欢大促。我的身边有女朋友在。可以说，去美国，去伊利诺伊，去芝加哥，去香槟城，都是因为她的缘故。然而今年，我却已然成为孤家寡人。只有Lv妹妹还能陪我聊几句。

我不知道如何用语言来表述这种感受，我能感觉到的，是一种孤独，是一种无奈，也是一种遗憾。我曾经是那么努力地去爱她，却终究敌不过。我一看到感恩节这几个字，就仿佛又回到了去年的今天，当时的欢乐，当时的爱情，当时的一切，都还历历在目，都还在我耳边回想。迎着陆家嘴今天的风，我都还能感觉到密歇根湖的那种要把人吹走的无情。我今天在朋友圈写到：

Nothing’s gonna change the decision I made.
是的，分开的决定是我最终做下的。我心意已决，我不会回头，但这不代表我不难过。我为我自己难过，也为曾经的感情难过。在这个感恩节，我除了感恩父母，感恩亲友，感恩那些需要我感恩的人之外，没有再别的了。

这，是一个只剩下感恩的感恩节。
