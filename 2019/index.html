金融狗在假装码农

Jan 31, 2019

这两天不知道发生了什么情况，商城的其他页面都还挺正常的，唯独这个会员登录的页面，怎么也载不出来，显示一个空白页，也没有任何的报错。正好趁这个机会，去巩固了一下python的内容，以后再拿其他的语言来弄这个商城网站吧，或者重新刷过。也许是我换login-bg出现的问题吧。需要再反复检查一下。

Jan 25, 2019

通过两天的折腾，把vps洗白刷新后，终于把商城上线了。地址为store.kenny4ever.com。

目前刚搭了框架，还只是把模板登上去了而已，后期要一步步改成自己的内容。另外，也还没有测试支付的东西，所以，还有很多要做。另外，使用的是opencart的商城模板，不过是国内公司汉化定制过的，先试试看，不行再换原版。

这次搭建商城，更多的是为了将来品牌上线做准备。

Jan 22, 2019

经过一晚上折腾，域名基本已经好了。这次索性将所有的内容都从coding搬到github了。对服务器在国内的repo还是有点害怕。

另外，发现github的强制https竟然还是会被报错，说是没加密，所以我还是动用了Cloudflare的DNS再加Https了。Porkbun的免费https就等到之后自己用服务器的时候再说吧。

也有考虑Vultr的，不过最近有点忙，而且要自己搭git，麻烦了点。

PS：换域名期间，如果发现部分链接打不开，可能是域名没更改。此项已经为todo项。

Jan 21, 2019

已经准备好了，新域名马上准备上线了，kenny4ever.com。

博客首页也会变成：blog.kenny4ever.com

有没有感觉像个正经网站多了。还是把.com顶级域名用起来比较靠谱。blog的名字也改了，因为，K先生已经不是一个人啦~~

最近两个月，原域名应该会打算弃用了，虽然没到期，但就用来各家搜索引擎搞些替换等等，CNAME之后应该就登陆不了了，也懒得做什么302跳转了。不过，到了三月下旬，原域名也到期了，那就直接转入新域名了。

如果您还在浏览我这个不怎么靠谱的博客主，那就请您更新下网站，谢谢啦。
